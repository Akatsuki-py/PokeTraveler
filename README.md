## PokeTraveler

## 方針
とりあえず複雑な戦闘、イベントは考えずポケモンGSC風の世界を探索できるゲームを目指すことにする
特定のマスに行ったら特定のキャラが話しかけてくるぐらいのイベントはやりたい
